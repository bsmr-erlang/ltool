## Installation

Just add it to your ``rebar.config`` deps:

```erlang
  {deps, [
    ...
    {ltool, ".*",
      {git, "git@github.com:lfe-rebar3/ltool.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar3 compile
```
